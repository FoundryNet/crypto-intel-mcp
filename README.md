# Crypto Intel MCP

Part of the **FoundryNet Data Network**.

Crypto market intelligence for agents: free `price` + `market_overview`, plus paid
`price_history`, `whale_alerts`, `defi_overview` and `anomaly_scan`. Data from
CoinGecko + DeFiLlama + the alternative.me Fear & Greed index (all keyless),
refreshed every 15 minutes into a standalone Supabase project.
